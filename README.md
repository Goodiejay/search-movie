# 🎬 Movie Viewer

A simple command-line Python application that searches for movies using the TMDb API.

## Features

* Search movies by title
* Search movies by genre
* View ratings, release year, language, and plot

## Tech Stack

* Python
* Requests
* TMDb API

## Run

```bash
pip install requests
python movie.py
```

## Preview

```text
1. Search by movie name
2. Search by genre
3. Exit
```
