# 🎬 Movie Viewer

A simple command-line Python application that searches for movies using the TMDb API.

## Features

* Search movies by title
* Search movies by genre
* View ratings, release year, language, and plot

## Tech Stack

* Python
* Requests
* python-dotenv
* TMDb API

## Setup

1. Clone the repo and install dependencies:

pip install requests python-dotenv


2. Create a `.env` file in the project root and add your TMDb API key:

TMDB_API_KEY=your_api_key_here


3. Run the app:

python movie.py
